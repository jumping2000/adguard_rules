# AdGuard

1. Login into AdGuard
2. Navigate to Filters
3. Click Add filter
4. Enter name
5. Copy this URL: https://github.com/jumping2000/adguard_rules/raw/master/adguard-facebook.txt
6. Paste the URL in the Edit box and click on Add filter
